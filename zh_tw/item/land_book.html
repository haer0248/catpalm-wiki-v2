# 領地群組之書

保護好自己的家當，避免遭小偷。而且苦力怕還炸不壞～真是太棒了～
可以綁定無限個領地產生器！
怪物不會受到保護、只保護在領地內誕生的動物。

# 創建新的

慣用手持未啟用的領地群組之書按右鍵，會變成已啟用的領地群組之書。

# 產生領地

慣用手持已啟用的領地群組之書對準領地產生器右鍵，可以寫入。

未寫入已寫入

如果領地產生器已經被寫入，需要先拆除才可重新使用，領地的擺法與烽火台在新視窗中開啟相同：

頂層方塊為領地產生器
下四層可以是領地小能量、領地中能量、領地大能量混搭
放滿四層滿獲得額外+20%能量點，能量點轉換公式：

能量點數
小1
中9
大81
沒有四層滿區塊半徑=無條件進位(平方根(總點數)/2)-1
四層滿區塊半徑=無條件進位(平方根(總點數x1.2)/2)-1

實際面積為總區塊面積=平方(區塊半徑+1+區塊半徑)
不會覆蓋比自己還早搶佔的領地：

# 獲取領地

如果不慎遺失了已啟用的領地群組之書，依然有方法取回。
慣用手持未啟用的領地群組之書對著領地產生器右鍵，可將已寫入的領地群組還原到書中。

# 更改名稱

將已啟用的領地群組之書在鐵砧在新視窗中開啟改成想要的名稱，領地群組的名稱會跟著改變。
可輸入最多36個任意字元。
由於快取機制，部份分流需要等待最久30秒新名稱才會生效。

# 群組之書

第一、二頁為說明，第三頁開始(ALL)=及全部權限，格式為玩家=權限清單。
玩家可以是暱稱或名稱，但編輯完成後一律自動轉換為暱稱。

#最大限制200名玩家
#格式:
#  玩家名稱=權限清單
#權限:
#  A 盔甲座
#  B 船
#  D 破壞方塊
#  E 實體互動
#  F 展示框
#  G 畫框
#  H 傷害實體
#  I 方塊互動
#  L 閱讀講座書
#  M 礦車

#  P 放置方塊
#  R 擁有者
#  S 生成實體
#  T 觸發機關
#  U 水桶
#  V PVP
#  Y 飛行

(ALL)=
你的玩家暱稱=ABDEFGHILMPRSTUVY

讓全部玩家都能飛行：

(ALL)=Y
你的玩家暱稱=ABDEFGHILMPRSTUVY

讓 I_PLAYER 玩家擁有全部的權限：

(ALL)=
你的玩家暱稱=ABDEFGHILMPRSTUVY
I_PLAYER=ABDEFGHILMPRSTUVY

每行的第一個字元為 # 表示為註解並不參與內容
例如將下方行

(ALL)=A

註解掉

#(ALL)=A

更新後此行會自行消失

# 活躍

以下行為會增加活躍度：

  • 實體貿易，每次+100
  • 方塊破壞、方塊放置、方塊更新，每次+40
  • 物品撿起、物品丟棄、容器拿取、容器放入，每次+20
  • 實體傷害，每次+10
  • 貓糧玩家存在，每秒+4
  • 玩家存在，每秒+1

每個區塊的活躍度上限為200,000，領地內活躍度每5,000可以保留1點領地能量。
間隔每12個月會檢查一次可以保留的領地能量點數，若總體低於20%會全部損壞，否則：

  • 未能被保留的領地能量將被損壞
  • 沒有任何能保留的領地能量將連同領地產生器一起損壞

指針朝向已激活的領地產生器會顯示從0~999%的活躍度參考，
公式=最小值(999,無條件捨去(當前總活躍度/((當前時間-上次檢查或創建時間)/6個月時間)/5000.0/當前總能量點數x100.0))
通常來說，只要超過100%都是安全無損傷的。
